Reveal a cell:
```
guess(row, col)
```
Start a new game:
```
newgame(size)
// board width & height = size
```
Auto play:
```
autoguess()
// random guess from remaining
```

Example:
```

-   -   -   -

-   -   -   -

-   -   -   -

-   -   -   -


> guess(2,3)

-   -   -   -

-   -   2   -

-   -   -   -

-   -   -   -

> guess(1,1)


    1   -   -

    1   2   -

1   2   -   -

-   -   -   -

> 

```
